# DeGrow HTML Templates

These templates are intentionally English-only for the first backend iteration.
They use double-brace placeholders so they can be rendered later by any backend
template engine, for example Handlebars, Nunjucks, Liquid, or a custom renderer.

Common variables:

- `{{app_name}}`
- `{{user_name}}`
- `{{support_email}}`
- `{{cta_url}}`
- `{{current_year}}`
